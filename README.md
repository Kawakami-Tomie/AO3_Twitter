# AO3_Twitter
使用CSS+html，在AO3实现twitter发帖和私信界面，可发帖、评论、自定义头像、帖子图片、显示时间、显示引用、点赞；有私信界面，可发图、回复文字、回复图片、显示已读等等...</br>
<p></p>
运行效果可点击AO3<a href="https://archiveofourown.org/works/74024686/chapters/193113396" rel="nofollow">FallinloveinOctober</a>下图为示例</br>
